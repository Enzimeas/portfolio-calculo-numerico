Interpolando os pontos de uma folha

Nessa ativiade, foi pedido que cada aluno trouxesse uma folha de árvore, uma peculiaridade da natureza é que muitas coisas são simétricas, se voce cortar uma folha ao meio e projetar a sua metade em um plano cartesiano, a sua borda tem o comportamento parecido de uma função. Com base nisso, foi pedido para que os alunos usassem uma régua para traçar 5 pontos usando as medidas reais da folha, após coletar esses dados, o aluno deve utilizar o código que tem a implementação do Método de Newton para achar um polinomio que interpola esses pontos, depois calcular o erro comparado com os valores reais.
